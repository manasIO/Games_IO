# Games.IO 🎮

A collection of browser-based games — no install, no server, just open and play.

## Games

| Game | Description | File |
|------|-------------|------|
| 🔫 **Shooter.IO** | Top-down shooting game | `Shooter_io/shooter.html` |
| 🟩 **Wordle.IO** | Guess the 5-letter Word of the Day in 5 tries | `Wordle_io/wordle.html` |

## How to Run

Open any `.html` file directly in your browser:

```bash
open Shooter_io/shooter.html
open Wordle_io/wordle.html
```

Or serve everything locally:

```bash
npx serve .
```

Built with vanilla HTML, CSS, and JavaScript — zero dependencies.
